eth.limo Example On-Chain Webpage

This is a demonstration of a base64 encoded web page stored on chain via the DataURL spec for ENS content hashes. Assets that are too large to store inline, such as fonts, are externally linked to a CDN. In the future there may be Ethereum RPC compatible chains with extremely cheap SSTORE operations and large transaction sizes that could enable next generation archival storage solutions. This example was created to be a minimal complete example of an on-chain webpage.
